## 📌 Problem Naming Convention

To keep this repository organized, all Java code files follow a specific numbering pattern to represent chapters and sub-problems. 

The format used is: **`Problem[Chapter_Number][Sub_Problem_Index]`**

### 🔍 Examples:
* **`problem10`** -> Represents **Problem 1 (a)**
* **`problem11`** -> Represents **Problem 1 (b)**
* **`problem21`** -> Represents **Problem 2 (a)**
* **`problem22`** -> Represents **Problem 2 (b)**
